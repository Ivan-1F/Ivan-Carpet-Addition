## 规则列表

### 自定义版本 (customVersion)

设置客户端服务器列表中显示的版本

设置为 '_' 来禁用

![screenshot](https://raw.githubusercontent.com/Ivan-YFw/Ivan-Carpet-Addition/fabric-1.15.2/screenshots/customVersion.png)

- 类型: `String`  
- 默认值: `_`  
- 参考选项: `1.19.6`
- 分类: `ICA`, `EXPERIMENTAL`
